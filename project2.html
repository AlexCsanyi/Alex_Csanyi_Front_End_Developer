Overview

A hotel website design built using React. A landing page with banners and cards, and a map and a rooms page with a filter fnctionality. I used the project with the main goal to learn how to use the context API to manage the data.

Content Structure

A very straightforward page layout. There are only 3 page types. From the landing page site visitors can go to the "rooms" page or go to one of the featured single room page, from the "rooms" page they can visit the single room page for each room type.

Functionality / Features

Users can have useful information about this fictional hotel and its servcies; on the rooms page they can filter the search results; site visitors have several options. I used styled components, mapbox with react-map-gl and react-icons in this project.
