Note 360.



Intro :

A note taking web app that is similar to Evernote

Technologies :

React, Firebase, Material UI

Status :

Working on adding "tagging" feature

I added React Quill so you can note text, this includes headers and paragraphs, but can also include bullets or numbered lists. Any supporting examples or media. The most common examples include images, videos, code snippets, graphs/charts, citations, etc. It is a single columned app, that is, the note-editor holds information inside a single text area, with all content vertically stacked on top of one another.
